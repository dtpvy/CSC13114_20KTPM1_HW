# Todo App

## Ver 1 - Get data from local storage

1. Vite

```bash
  yarn create vite ver-1 --template react-ts
  cd ver-1 && yarn
```

2. Run dev

```bash
  yarn run dev
```

## Ver 2 - Use redux

1. Vite

```bash
  yarn create vite ver-2 --template react-ts
  cd ver-2 && yarn
```

2. Install redux

```bash
  yarn add @reduxjs/toolkit react-redux
```

3. Run dev

```bash
  yarn run dev
```
